# @anon-aadhaar/react-native

React Native library for the Anon Aadhaar protocol

## Installation

```sh
npm install @anon-aadhaar/react-native
```

## Usage

```js
import { multiply } from '@anon-aadhaar/react-native';

// ...

const result = await multiply(3, 7);
```
